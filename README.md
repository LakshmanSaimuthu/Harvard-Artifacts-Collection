# 🏛️ Harvard Artifacts Collection — Data Engineering & Analytics Project

## 🚀 Project Summary

Built an end-to-end data pipeline and analytics dashboard using the **Harvard Art Museums API**, demonstrating real-world skills in **data engineering, SQL analytics, and interactive dashboard development**.

The application extracts museum artifact data, transforms it into structured datasets, stores it in a relational database, and enables interactive analysis through a Streamlit web interface.

This project showcases practical experience in building **production-style data workflows**.

## 📊 What the Application Does

✔ Collects thousands of museum artifacts dynamically from API
✔ Converts nested JSON into normalized SQL tables
✔ Stores structured data in MySQL automatically

✔ Provides an interactive dashboard for exploration

## ▶️ How to Run

```bash
git clone https://github.com/LakshmanSaimuthu/Harvard-Artifacts-Collection.git
cd Harvard-Artifacts-Collection
pip install -r requirements.txt
streamlit run app.py
```

Create a MySQL database named:

```
harvardfinalproject
```

Update credentials inside `app.py` before running.
